Printing a map in XMind is easy. Just follow these steps:

  1. Select a map.
  2. Open the print preview dialog by either
    • Choose 'File > Print' on the menu.
    • Use the shortcut 'Ctrl+P'.
  3. Change the Page attributes: Margin size and unit, Header/Footer text.
  4. Click 'Next' and choose the printer you wish to print to.
  5. Click 'OK'.

Note:

  1. XMind does not support printing multiple maps together in one sheet of paper even if they are in the same workbook. You will have to print each map individually.
