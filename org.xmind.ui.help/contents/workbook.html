Collaborative Minds

Every XMind file is considered a workbook. There can be multiple maps in one workbook, and every map is a separate sheet within the workbook. You can easily build multiple maps and modify them within an XMind workbook. Here's how:

To build multiple maps within a workbook, use one of these three methods:

  1. Double-click on the blank space beside the sheet name at the bottom of the editor to build new map.
  2. Click 'sheet' icon on the toolbar.
  3. Use shortcut 'Ctrl+T'.

To modify sheet:

  1. Select a sheet from the bottom of the editor.
  2. Right-click on the sheet name with your mouse to open context menu.
  3. From the context menu you can make the following changes:
    • Rename this sheet.
    • 'Save Map as...' let you save the current sheet as a separated workbook.
    • Delete the current sheet from the workbook.


Note: All maps in a workbook are separate from each other. They can have different structures, wallpapers, themes, and so on.
