Collaborative Minds

The Merge maps can help you to organize the information from more than one map by merging two maps together. In this example, Map A will be merged with Map B.

  1. Open Map A.
  2. Choose 'Tools > Merge..' on the menu to Open the merge dialog.
  3. Select Map B.
  4. Click 'OK' and you can see the merged map.

Merge Rules

  1. Map A and Map B should both be opened in XMind.
  2. There is no difference between merging Map A to Map B, or merging Map B to Map A.
  3. Map A and Map B can be in different workbooks.
  4. Topics with the same text content in the same level under the same parent will be treated as identical topics; identical text inside the corresponding topic on Map A will be replaced by the text on Map B.
  5. Topics with different text contents will be treated as different topics. If a topic on Map B does not exist in Map Q, then it will be copied to Map A under the corresponding parent.
  6. Pressing "Ctrl+Z" can undo the merge process.
