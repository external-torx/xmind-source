XMind Pro supports exporting to a variety of file formats. You can export maps to MindManager files, PDF, MS Word, Powerpoint, image file formats, and more. Here's how:

To Export to PDF(Map):

  1. Choose "File > Export" on the menu.
  2. Select "PDF(Map)" in the export dialog.
  3. Click "Next" to continue.
  4. Choose the location and name the file after clicking 'Browser'.
  5. Change the settings: Page Size, Orientation, and Resolution.
  6. Click 'Finish' to end this process.

To Export to PDF:

  1. Choose "File > Export" on the menu.
  2. Select "PDF Document" in the export dialog.
  3. Click "Next" to continue.
  4. Choose the location and name the file after clicking 'Browser'.
  5. Change the settings following and see the changes in the preview dialog: 'Overview', 'Separate Overview', 'Notes', 'Floating Topics', 'Labels', 'Markers', 'Image', 'Relationships', 'Hyperlinks'.
  6. Click 'Finish'to complete the export process.

To Export to Mindjet MindManager Map:

  1. Choose "File > Export" on the menu.
  2. Select "Mindjet MindManager Map" in the export dialog.
  3. Click "Next" to continue.
  4. Choose the location and name the file after clicking "Browser".
  5. Click "Finish" to end this process.

To Export to Image:

  1. Choose "File > Export" on the menu.
  2. Select "Image" in the export dialog.
  3. Click "Next" to continue.
  4. Choose the location and name the file after clicking "Browser".
  5. Select the Image Type: JPG, PNG, GIF or BMP.
  6. Click "Finish"to complete the export process.

To Export to RTF:

  1. Choose "File > Export" on the menu.
  2. Select "RTF File" in the export dialog.
  3. Click "Next" to continue.
  4. Choose the location and name the file after clicking "Browser".
  5. Click "Finish"to complete the export process.

To Microsoft Word:

  1. Choose "File > Export" on the menu.
  2. Select "Word 97-2003 document" or "Word 2007 document" in the export dialog.
  3. Click "Next" to continue.
  4. Choose the location and name the file after clicking "Browser".
  5. Change the settings if needed and see the changes in the preview dialog: "Overview", "Separate Overview", "Notes", "Floating Topics", "Labels", "Markers", "Image", "Relationships", "Hyperlinks".
  6. Click "Finish"to complete the export process.

To Microsoft PowerPoint:

  1. Choose "File > Export" on the menu.
  2. Select "PowerPoint 97-2003 Presentation" or "PowerPoint 2007 Presentation" in the export dialog.
  3. Click "Next" to continue.
  4. Choose the location and name the file after clicking "Browser".
  5. Change the settings if needed and see the changes in the preview dialog: "Overview", "Separate Overview", "Notes", "Floating Topics", "Labels", "Markers", "Image", "Relationships", "Hyperlinks".
  6. Type the "Footer text".
  7. Click "Finish"to complete the export process.

Note:

  • The preview is an example for your reference. The appearance of your map may vary from the examples presented here.
  • Exporting to Microsoft Word and PowerPoint is only supported for Windows Operating system at present. You must have Microsoft Word or PowerPoint installed to view the exported files.
