Wallpaper is a background image that can be used to customize the look of your mind map. You can use one of the existing wallpaper options in XMind, or import your own image to use as wallpaper. Here's how:

To Add the wallpaper to your map:

  1. Select the map.
  2. Open Properties View.
  3. Click 'Wallpaper...' and select the wallpaper you like in the pop-out dialog.
  4. OR, click 'Select local image..' and select your own image from local file directory.

Wallpaper list dialog

To Modify or delete the existing wallpaper:

  1. Select map.
  2. Open Properties View
    • Click the delete icon in the properties view to delete current wallpaper.
    • Change wallpaper opacity by imputing a number or moving the slider.
