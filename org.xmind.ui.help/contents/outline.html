Collaborative Minds

Outline is a tree-like view in XMind. Open outline view and you can see all elements of the current map or wookbook. Additionally you can modify some of the map's attributes here directly:

  • Topic: You can drag and drop a topic from outline view into the XMind editor. That topic will be copied into current map. You can directly double-click a topic in the outline to edit it.
  • Marker: You can see Priority, Smiley, Task Complete, and Flags markers here, if some topics contain them.
  • Hyperlink: You can see the hyperlink URL if a topic use one.

Outline View Toolbar

Outline has its own toolbar.

  • Show Workbook: Click to show all maps in this workbook.
  • Show Current Map: Click to show the current map only.
