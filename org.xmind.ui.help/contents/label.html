A label is a small text tag that is attached to topics and typically used for simple annotation and categorization. Added labels will be listed in the advanced filter view. You can filter the topics within the map by label.

You can easily create, edit, and delete existing label. Here's how:

To add a label:

  1. Select a topic.
  2. You have three options to open label field:
    • Click the label icon on toolbar.
    • Right-click with your mouse to open the context menu and click 'label'.
    • Use the shortcut key 'F3'.
  3. Type the label text in the field below the selected topic.
  4. Click 'Enter'.

Every topic can have multiple labels. Just use comma to separate them. Ant the labels will be in alphabet order.

To edit a label:

  1. Select the topic.
  2. Open the label field.
  3. Edit the existing label.
  4. Click 'Enter'.

To delete a label:

  1. Select topic.
  2. Open the label field.
  3. Delete this existing label.
  4. Click 'Enter'.

Note: You cannot create, edit, or delete a label for multiple topics at the same time.
