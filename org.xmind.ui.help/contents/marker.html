Marker

Markers are image icons used to represent attributes on a map, and are widely used in project management, planning, and other organizational activities. XMind provides a wide array of pre-made markers and allows you to add your own images to the Gallery to be used as markers. You can also import marker group files from other XMind users as your own marker group, or export your own marker group to others. Here's how:

To add a marker(s):

  1. Select the topic(s).
  2. Open the Markers View.
  3. Click the marker you want to add to your map.
  4. This marker will be added into selected topic(s).

To delete existing marker:

  1. Select the topic.
  2. Select the marker.
  3. Choose one of the following methods to delete the marker:
    • Click the 'Delete' key on keyboard directly.
    • Right-click the marker with your mouse and select 'delete'.

To use your own markers:

  1. Open Markers View.
  2. Click 'Show Marker Manager' icon on view toolbar to open add dialog.
  3. Click 'Add' beside 'Marker groups' field, and name the new group.
  4. You can remove, rename, move up, or move down the group added.
  5. Select the new marker group, click 'Add' beside 'Markers' field.
  6. Select the image(s) you need, name it, and click 'open' to add into XMind.
  7. You can remove, rename, move up, or move down the added markers.
  8. Click 'Apply' to finish this process.
  9. You can see your own marker group at Markers view.

 

To import other's markers

  1. There are two manners to open the Marker Package import dialog:
    • Open Markers View, and click "Import Marker Group" in the view toolbar.
    • Choose "File > Import" in the menu, and select "Marker Package".
  2. You can now import Marker Group file('.xmp') or local images from th selected folder.
  3. Click "Finish".

To Export your own marker group

  1. Choose "File > Export" in the menu
  2. Select "Marker Package"
  3. Select save path, and click "Finish"

Legend

A Legend is a floating box within your map that describes all of the map markers and their specific meanings.

To show legend:

  1. Select the map.
  2. Open the Properties View.
  3. Click on the 'Show Legend' box to toggle the legend on.
  4. Move you mouse to place legend on the map.
  5. You can rename any of the markers by double-clicking the current name on the legend.

Note: You can move the legend to any position on the map by dragging the legend with yoru mouse.
