Hyperlinks in a topic can be used to connect maps to websites, folders, or files such as Excel or PDF documents. Each topic can be given a URL that can be presented as a hyperlink (multiple URLs cannot be associated with a single topic).

You can easily create, open, modify, and delete existing hyperlinks, and also add local file address as hyperlinks. Here's how:

To Create a hyperlink:

  1. Select a topic.
  2. Open the hyperlink dialog by using one of these methods:
    • Choose "Modify > Hyperlink" on the menu.
    • Click hyperlink icon on the toolbar.
    • Press 'Ctrl + H' (or Command + Shift + H for Mac users).
    • Right click with your mouse to open context menu, and select "Hyperlink"
  3. There are three kinds link in XMind.
    1. "Web": Enter the URL(for example: http://www.XMind.net).
    2. "File": Add a local file or folder link. Now, you can choose to use Relative link or Absolute link.
    3. "Topic": Link a topic in current workbook.

If you add local file as relative hyperlink, XMind will ask you to save the file first.

To Open a hyperlink:

  1. Select a topic.
  2. There are two ways to open hyperlink in this topic:
    • Open from menu.
      • For Web link: Select "Modify > Open Hyperlink" on the menu.
      • For local file link: Select "Modify > Open File".
      • For local folder link: Select "Modify > Open Folder".
      • For Topic link: Select "Modify > Go To Topic".
    • Right-click mouse and open topic context menu, select "Open Hyperlink", "Open File", "Open Folder", or "Go To Topic".

To Modify a Hyperlink

  1. Select the topic that contains a hyperlink
  2. Right-click with your mouse and open context menu, then select "Hyperlink".
  3. Modify the link in the dialog.
  4. Click "ok" to finish.

To Delete a hyperlink

  1. Select the topic with hyperlink.
  2. Right-click to open context menu, select "Cancel Hyperlink".
