In XMind Pro, you can lock your XMind file with a password. Then, you can have a high security level to protect your file. Here's how:

To Set the password:

  1. Open or draw a map.
  2. Choose "File > Security" on the menu.
  3. You can see a set password dialog.
  4. Type your password twice, and click "OK" button.


To Reset/Remove the Password:

  1. Open a map which has the password
  2. Input the password and click "OK" to open the file
  3. Choose "File > Security" on the menu.
  4. Input the Old Password.
  5. Now, there are two options:
    • Leave the "New Password" field blank, and click "OK": Remove the Old Password
    • Input the new password twice in two fields: Reset the Password

Note: Please keep your password carefully. If you forget the password, that file will never be opened.
