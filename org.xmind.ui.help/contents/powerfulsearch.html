A powerful search can help us to find the targeted information soon. In XMind, you can search a certain text in all XMind workbooks which are opening. Here's how:

  1. Open Search View: "Tools > Search Workbooks"
  2. Now, typy the content you want at the field
  3. Select the advanced options:
    • Case sensitive: If caring about the Case
    • Only the active editor: If searching in the active editors or all workbooks
    • Topics: Enabling this will search all topics.
    • Boundaries: Enabling this will search all boudaries' title.
    • Relationships: Enabling this will search all relationships' title.
    • Labels: Enabling this will search all labels.
    • Notes: Enabling this will search all notes.
  4. Click "Search" button and begin to search
  5. All search results will be displayed as a tree structure. We can see the full content, and the full path.
