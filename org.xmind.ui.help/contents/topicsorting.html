Topic Sorting can help us find out useful information, inspect work easily, esp. while we are facing a big mind map. Here's how to use this useful feature in XMind:

  1. Select a topic. Now you can sort its subtopics via Priority, Topic Title, or Modified time
  2. There are two ways to sort topics:
    • From Menu: "Modify > Sort > By Priority/Title/Modified time"
    • From Context Menu: "Right click mouse > Sort > By Priority/Title/Modified time"

Notes:

  1. XMind sorts all subtopics of your selected topic.
  2. If you select "By Title", topics will be sorted alphabetically.
